About me

A curious mind from Kathmandu, coding in Finland

I'm Rajan Bam, a Computer Science student at LUT University in Lappeenranta, Finland. I completed my higher-secondary education (+2, Science stream) at Morgan International College, Kathmandu, where my fascination with how the web works first began.

I love the moment a plain HTML page comes alive with CSS and JavaScript. This portfolio is the project for the Software Development Skills: Front-End course — every section here is hand-built to practice semantic markup, responsive layouts, Flexbox, CSS Grid and interactive JavaScript, without relying on frameworks.

  • Clean, semantic & accessible HTML
  • Mobile-first responsive design
  • Vanilla JavaScript, no frameworks

It's my course project for LUT's Software Development Skills: Front-End module — a single-page responsive portfolio built with plain HTML, CSS and JavaScript to demonstrate what I've learned.

Semantic HTML5, modern CSS (custom properties, Flexbox, CSS Grid, media queries) and vanilla JavaScript for the menu, theme toggle, slideshow, accordion and scroll animations. No frameworks or libraries.

Yes — it's mobile-first and tested from 320px phones up to large desktops. The navigation collapses into an animated hamburger menu on small screens.

Clone the repository and open index.html in a browser, or serve it with any static server (for example python -m http.server). Full steps are in the README.

The site is deployed with GitHub Pages straight from this repository, so any push to the main branch updates the live version.
