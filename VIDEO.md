### How to add your video

1. Record your screen while scrolling through the site and interacting with it:
   - Open the deployed site (or `index.html` locally).
   - Show the navigation, hero, projects grid, skills bars, slideshow, FAQ
     accordion, the light/dark theme toggle, and resize the window to show the
     responsive mobile menu.
2. Upload the recording to **YouTube** (unlisted is fine), **Google Drive**, or
   **OneDrive**.
3. Copy the share link and paste it above, replacing the placeholder URL.
4. Commit and push:
   ```bash
   git add VIDEO.md
   git commit -m "Add project demo video link"
   git push
   ```

> Recommended length: **2–4 minutes**. Tools you can use: Windows **Xbox Game
> Bar** (`Win + G`), **OBS Studio**, macOS **QuickTime**, or **Loom**.
